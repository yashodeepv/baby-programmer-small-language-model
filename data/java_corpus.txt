USER: What is the difference between ArrayList and LinkedList?
ASSISTANT: ArrayList uses a dynamic array. Random access O(1), insert/delete O(n). LinkedList uses nodes. Insert/delete O(1) at head/tail, random access O(n). Use ArrayList for most cases.

USER: What is recursion?
ASSISTANT: Recursion is when a method calls itself. Every recursive method needs a base case to stop. Example: factorial(n) = n * factorial(n-1), base case: factorial(0) = 1.

ASSISTANT: An algorithm is a step-by-step procedure to solve a problem. It has a defined input, clear steps, and produces a correct output. Examples: sorting algorithms, search algorithms.

USER: What is a data structure?
ASSISTANT: A data structure organizes and stores data for efficient access and modification. Examples: array, ArrayList, HashMap, Stack, Queue, LinkedList, Tree.

USER: What is object-oriented programming?
ASSISTANT: OOP is a programming paradigm that organizes code into objects with state (fields) and behavior (methods). The four pillars are encapsulation, inheritance, polymorphism, and abstraction.

USER: What are the four pillars of OOP?
ASSISTANT: Encapsulation: hiding data. Inheritance: child class extends parent. Polymorphism: same method, different behavior. Abstraction: hiding implementation details with interfaces or abstract classes.
